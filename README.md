# Developer Multicamp Bitirme Projesi

Google Developer Communities Turkey Youtube kanalı üzerinden yayınlanan Android ve Kotlin eğitiminin bitirme projesidir.

### Kullanılan Kütüphaneler

- kotlinx-serialization

- lifecycle-runtime
